## Slack Integration

### Prerequisites

* Slack workspace admin access

* Slack API token [How to get slack API bot token](https://api.slack.com/tutorials/tracks/getting-a-token)

* Target Slack channel name

### Configuration Steps

1. Navigate to **Settings > Alerts** in DefendStack platform

2. Locate the Slack section

3. Click the **Configure** button next to the Slack icon

4. Enter the following details:

   * Slack API Token

   * Channel Name (where alerts will be posted)

5. Click **Add** to save the configuration<img height="200" src="/slack.png" />

## Jira Integration

### Prerequisites

* Jira admin access

* Jira access token

* Project key

* User email associated with Jira account

* Jira application URL

### Configuration Steps

1. Navigate to **Settings > Alerts** in DefendStack Platform

2. Locate the Jira section

3. Click the **Configure** button next to the Jira icon

4. Enter the following details:

   * Jira Application URL

   * Access Token

   * Project Key

   * User Email

5. Click **Add** to save the configuration<img height="200" src="/jira.png" />
